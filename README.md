# MAP GENERATOR

## GOAL
Implement the wave collapse functions to generate maps with ASCII Characters

### Characters
Theses are the used characters to build the map

```
	"╣", "╠", "║", "═", "╝", "╚", "╗", "╔", "╩", "╦", "╬"

```
